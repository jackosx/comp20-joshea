Desciption: This assigment is a site that locally stores messages input by the user, along with the time they were entered. The user types the message into the text field, and presses the enter key to save it. It is then displayed below the field, along with all previous messages. Anytime the page is loaded the previous entries are retrieved from local storage and displayed as well.

Completion Time: ~90 Minutes
